# 🛒 BudgetGenie AI

**BudgetGenie AI** is a smart **buyer-based product recommender** that helps users find the best products under their budget. 💡💰

## Features
- Enter your budget and select your country 🌎
- Get top product recommendations based on **ratings** and **popularity** ⭐📈
- Trend score calculated using a combination of rating and number of reviews for smarter recommendations 🔍
- Easy-to-use interface built with **Streamlit** 🖥️

## How It Works
1. User inputs budget and country.
2. The system filters products under the budget in the selected country.
3. Products are ranked by **trend score** to recommend the top 3 items.

## Tech Stack
- Python 🐍
- Streamlit ⚡
- Pandas & NumPy 📊
